HTML 멀티미디어 삽입

object태그

-오디오, 비디오, PDF 등 다양한 멀티미디어 파일 삽입
-웹 문서 안에 다른 문서를 삽입할 때도 사용 가능
-기본형

embed태그

-HTML 초기 버전부터 사용: 대부분의 브라우저에서 사용 가능
-오디오, 비디오, 이미지 등 다양한 멀티미디어 파일 삽입

audio태그

-오디오, 비디오 파일을 삽입
-HTML4까지는 웹 브라우저에서 멀티미디어 파일을 삽입 후 재생 시에 플러그인 프로그램 설치가 필수
(플러그인 프로그램 ex. 곰플레이어)
-HTML에서 배경음악이나 효과음 등 오디오 파일 삽입

video태그

-비디오 파일을 삽입

audio, video태그 속성

-controls: 플레이어 화면에 컨트롤 바를 표시
-autoplay: 오디오나 비디오를 자동으로 실행
-loop: 오디오나 비디오를 반복 재생
-muted: 오디오나 비디오의 소리를 제거
-width, height: 플레이어의 너비와 높이를 지정
-poster="파일이름"
해당 속성이 지정된 비디오가 재생되기 전까지 화면에 표시될 포스터의 이미지를 지정
