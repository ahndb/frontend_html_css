정의-표준화된 기술이나 규칙

중요성 -일관된 웹 경험 제공, 접근성 보장, 유지보수용이

접근성-모든 사용자가 '웹 사이트와 도구'에 '동등'하게 접근 할 수 있게 만드는 것

주요 기술
html 웹 페이지 기본 구조 내용 -HTML5
css 웹 페이지 디자인 (레이아웃, 애니메이션, 글꼴 등) -CSS3
javascript 웹 페이지 동적인 요소 추가 
dom 웹 페이지의 객체와 구조를 표현, js가 페이지와 상호작용 하도록 도와주는 인터페이스
accessibility guidelines 웹 콘텐츠를 모든 사용자들이 접근하고 이해할 수 있도록 하는 웹 접근성 가이드라인
